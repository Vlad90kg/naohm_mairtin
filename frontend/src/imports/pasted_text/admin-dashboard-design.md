Design a clean, modern Admin Dashboard for managing a GAA club website (Naomh Mairtin GFC). The dashboard should be simple, functional, and focused on content management rather than complex analytics.

Style:
- Clean and minimal UI
- Light background (white/light grey)
- Blue accents (#1E3A8A or similar)
- Card-based layout
- Clear typography and hierarchy
- Simple, professional design (not overly decorative)

---

LAYOUT STRUCTURE:

1. Sidebar Navigation (left side, fixed):

- Logo at top
- Menu items:
  Dashboard
  Events
  Lotto
  Sponsors
  Media (optional)
  Settings (optional)

- Simple icons next to labels
- Clean vertical layout

---

2. Top Bar:

- Page title (e.g., “Dashboard”)
- Admin user (top right)
- Optional logout button

---

MAIN DASHBOARD CONTENT:

3. Overview Cards:

Display 3–4 summary cards:
- Total Events
- Active Sponsors
- Latest Lotto Update
- Recent Activity (optional)

Cards should be simple and minimal

---

4. Quick Actions Section:

Buttons/cards for:
- Add New Event
- Update Lotto Results
- Add New Sponsor

Make actions clearly visible and easy to access

---

---

FEATURE SECTIONS:

5. Events Management:

- Table or card list of events:
  - Title
  - Date
  - Location

- Actions:
  Edit / Delete

- Button:
  “Add Event”

---

6. Lotto Management (IMPORTANT):

- Section for updating weekly lotto results
- Fields:
  - Week / Date
  - Results
  - Jackpot info

- Button:
  “Update Lotto”

- Keep very simple (fast updates required)

---

7. Sponsors Management (CRITICAL):

- List of sponsors displayed as cards or table

Each sponsor includes:
- Logo
- Name
- URL
- Tier (1, 2, or 3)

- Actions:
  Edit / Delete

- Button:
  “Add Sponsor”

---

8. Add/Edit Sponsor Form:

Fields:
- Sponsor Name
- Upload Logo
- Website URL
- Select Tier (1, 2, 3)

Important:
- Tier selection determines placement on website

---

9. Media Upload (Optional):

- Upload images for events/news
- Simple grid of uploaded files

---

---

FUNCTIONAL REQUIREMENTS:

- Focus on ease of use (non-technical users)
- Keep interface simple and intuitive
- No unnecessary complexity
- Prioritise:
  - Events
  - Lotto updates
  - Sponsor management
- No user/member management required
- Responsive layout

---

DESIGN RULES:

- Use cards and simple tables
- Avoid clutter
- Use clear spacing and alignment
- Make primary actions obvious (buttons)
- Keep navigation simple and consistent ------------------------ Do not include member login systems.
Do not add unnecessary analytics charts.
Do not overcomplicate forms.
Keep dashboard focused on content management only.
