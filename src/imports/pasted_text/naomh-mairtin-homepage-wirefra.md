Design a modern, responsive homepage wireframe for a GAA club website (Naomh Mairtin GFC), inspired by a mobile app UI with a clean, card-based layout.

Style:
- Primary color: deep blue (#1E3A8A or similar)
- White and light grey card backgrounds
- Gold/yellow accents for highlights (especially Tier 1 sponsors)
- Rounded corners, soft shadows, minimalistic UI
- Clean typography with strong headings and short text
- Consistent spacing and grid system

---

GLOBAL HEADER:

1. Tier 1 Sponsor Strip (top of page):
- Full-width horizontal bar
- Light background
- Display 3–5 sponsor logos evenly spaced
- Logos clickable
- Compact height (60–80px)

2. Main Navigation Bar (sticky, below sponsor strip):
- Club logo on left
- Menu items:
  Home, About (dropdown), Teams (dropdown), Events, Lotto, Shop, Membership, Sponsors, Contact, Safeguarding
- Clean alignment and spacing

---

HOMEPAGE SECTIONS (top to bottom):

3. Hero Section:
- Full-width blue background
- Centered club logo
- Heading: “Welcome to Naomh Mairtin GFC”
- Short supporting text
- Two buttons:
  - Primary: “View Events”
  - Secondary: “Membership Info”

---

4. Featured Announcement:
- Card-style block
- Title, short description, CTA button

---

5. Upcoming Events Section:
- Title: “Upcoming Events”
- Display 3 event cards:
  - Event title
  - Date
  - Location
- Card layout (inspired by mobile app)
- CTA: “View Full Calendar”

---

6. Latest News Section:
- Title: “Latest News”
- Card layout with:
  - Image
  - Headline
  - Timestamp
- 3–4 cards in grid

---

7. Teams Overview:
- Title: “Our Teams”
- Two main cards:
  - Adult Teams
  - Juvenile Teams
- Each card includes short label + CTA “View”

---

8. Sponsors Section:
- Title: “Our Sponsors”
- Tiered layout:
  - Tier 1: large logos (highlighted with subtle gold accent)
  - Tier 2: medium logos
  - Tier 3: grid of smaller logos
- CTA button: “Become a Sponsor”

---

9. Lotto and Shop Section:
- Two side-by-side cards:
  - Lotto:
    - Short description (weekly results)
    - CTA: “View Results”
  - Club Shop:
    - Short description
    - CTA: “Visit Shop”

---

10. Membership CTA Section:
- Short call-to-action text
- Button: “Become a Member”
- Links to external system (no login UI)

---

11. Footer:
- Compact layout
- Include:
  - Address + Eircode
  - Email contact
  - Navigation links (same as header)
  - Social media icons
- Clean and minimal design

---

FUNCTIONAL REQUIREMENTS:

- Header is sticky
- Tier 1 sponsor strip visible on all pages
- Card-based layout used consistently
- Avoid long text blocks
- No login functionality
- No duplicate sponsor sections
- Maintain clear visual hierarchy
- Design must match mobile app style (blue + white + clean cards)
